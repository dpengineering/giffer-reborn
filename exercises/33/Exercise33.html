NEW: Multiple Parameters
We can have as many parameters as we want in a function, separated by commas. This allows us to pass multiple pieces of information into the function to customize its behavior even further.

Here is an example a parameter having two parameters for blinking, onTime and offTime: 

                    void loop() {
                      blink2(500, 250); // Call blink2 with an onTime of 500 ms and offTime of 250 ms
                      blink2(1000, 2000); // Call blink2 with an onTime of 1000 ms and offTime of 2000 ms
                    }

                    void blink2(int onTime, int offTime)
                    { 
                      digitalWrite(2, HIGH);
                      delay(onTime); 
                      digitalWrite(2, LOW);
                      delay(offTime);
                    }

Hints

  1. Start by creating an additional parameter named repeatCount.
  2. Add a for loop to the blink2 function that repeats "repeatCount" times.
  3. Call blink2 with different values for repeatCount to see the effect.
  4. Your main loop should only have four lines of code...
