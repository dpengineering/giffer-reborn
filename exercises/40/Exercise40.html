Colorful Arrays

NEW: Arrays
Arrays are a way to store multiple values in a single variable. They are useful for grouping items together, even if they are not in a specific order. 

                    int reds[4] = {2,4,6,8};
                    int greens[3] = {3,5,7};
In the example above, we create two arrays that hold int values. The first array, `reds`, holds four values: 2, 4, 6, and 8. The second array, `greens`, holds three values: 3, 5, and 7.
NEW: Array Indexing
We can use arrays to group related items together. In this case, we can group our LED pin numbers into an array. This allows us to easily reference and manipulate multiple LEDs using loops. Each LED's pin number is stored at a specific index in the array, starting from 0. For example, if we have an array `ledPins` with values `{2, 3, 4, 5}`, `ledPins[0]` refers to pin 2, `ledPins[1]` refers to pin 3, and so on. By using loops, we can iterate through the array indices to control each LED in sequence or simultaneously. 

                    void blinkArray(int ledArr[], int arrLength, int t){
                        for (int i=0; i < arrLength; i++){
                          digitalWrite(ledArr[i], HIGH);
                        }
                        delay(t);
                        
                        for (int i=0; i < arrLength; i++){
                          digitalWrite(ledArr[i], LOW);
                        }
                        delay(t);
                      }
In the example above, blinkArray indexes through the ledArr array to turn on and off each LED in the array.
