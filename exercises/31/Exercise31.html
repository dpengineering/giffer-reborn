First Time Functioning

NEW: Defining a Function
A function is a block of code that can be called multiple times. Functions help to reduce repetition in code and make it more organized. You define a function once, and then you can call it whenever you need it.

Here is an example of a simple blink function in Arduino: 

                void blink2() {
                        // Function name: blink2
                        // Return type: void (does not return any value)
                        // Function body: Contains the code to be executed when the function is called

                        digitalWrite(2, HIGH); // Turn the LED on
                        delay(1000);           // Wait for 1 second
                        digitalWrite(2, LOW);  // Turn the LED off
                        delay(1000);           // Wait for 1 second
                }
NEW: Calling a Function
To use the function, you simply call it by its name followed by parentheses: 

                void setup() {
                        pinMode(2, OUTPUT); // Set pin 2 as an output
                }

                void loop() {
                        blink2(); // Call the blink2 function to blink the LED
                }
In this example, the blink2 function is defined once and can be called multiple times within the loop function. This helps to keep the code clean and avoids repetition.
