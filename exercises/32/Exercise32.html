Parametric Decay

NEW: Function Parameters
Functions can also take parameters, which are values you pass into the function to customize its behavior. Parameters allow you to make functions more flexible and reusable.

Here is an example a parameter "t" added to the blink2 function in Arduino: 

                    void loop() {
                      blink2(500); // Call blink2 with a delay of 500 milliseconds
                      blink2(1000); // Call blink2 with a delay of 1000 milliseconds
                    }

                    void blink2(int t)
                    { 
                      digitalWrite(2, HIGH);
                      delay(t); // Here we use the parameter t to set the delay time
                      digitalWrite(2, LOW);
                      delay(t);
                    }
