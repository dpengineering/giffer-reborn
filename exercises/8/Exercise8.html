Let's Loop

NEW: Multiple variables
blinkTime and count are both variables. You can have as many variables as you want, just be sure to give each one a unique name.
NEW: while(count <= 4)
A while loop will continue to execute the code inside its curly braces { } as long as the condition inside the parentheses ( ) is true. In this case, the code will keep executing as long as the variable count is less than or equal to 4.
Be careful to ensure that the condition will eventually become false, otherwise the loop will run forever!
NEW: count += 1;
This is a shorthand way of writing count = count + 1;. It takes the current value of count, adds 1 to it, and then stores that new value back into the variable count.
Other shorthand operators include: -=, *=, and /=.
