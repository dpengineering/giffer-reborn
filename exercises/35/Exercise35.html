NEW: Descriptive Function Names
We can make our code more readable by giving our functions descriptive names. Instead of naming a function blink2, we can name it blinkRightSideUp to indicate that it blinks the right side of the LEDs from bottom to top. 

                    void blinkRightSideUp(int onTime, int offTime)
                    { 
                      for (int ledNum = 2; ledNum <= 8; ledNum ++)
                      {
                        digitalWrite(ledNum, HIGH);   // turn the LED on (HIGH is the voltage level)
                        delay(onTime);                // wait for onTime milliseconds
                        digitalWrite(ledNum, LOW);    // turn the LED off by making the voltage LOW
                        delay(offTime);               // wait for offTime milliseconds
                      }
                    }
